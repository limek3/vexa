# Calendar unblock confirmation pass

## Что изменено

- Клик по объединённому заблокированному диапазону больше не разблокирует слот сразу.
- Теперь открывается отдельное подтверждение с датой и диапазоном времени.
- Разблокировка выполняется только после клика `Да, разблокировать`.
- Окно можно закрыть через `Отмена`, крестик, Escape или клик по затемнению.
- Во время сохранения подтверждение нельзя случайно закрыть кликом по фону.
