# ClickBook Master Minimal Pass 2

Цель прохода: вернуть визуальное направление ближе к прошлому premium-варианту, но оставить продуктовую логику сервиса записи и CRM для мастеров.

## Что изменено

- Убран чрезмерный warm/CRM-стиль из первого прохода.
- Дизайн-система переведена в более строгий минимализм: кремовый фон, белые карточки, тёмно-синий текст, коралловый акцент.
- Header стал компактнее: без большого декоративного hero, без лишней «продающей» подачи.
- Карточки стали спокойнее: меньше padding, меньше визуального шума, тоньше акценты.
- Главная страница стала ближе к рабочему premium-overview: короткие метрики, лента записей, тёмная premium-карточка ближайшей записи.
- Страница `/dashboard/today` упрощена: расписание, неделя, метрики и действия выглядят строже и минималистичнее.

## Файлы

- `components/workspace/master-ui.tsx`
- `app/globals.css`
- `app/dashboard/page.tsx`
- `app/dashboard/today/page.tsx`

## Направление дальше

Дальше все страницы нужно переводить не в «SaaS», а в этот стиль:

- меньше декоративных градиентов;
- больше воздуха и сетки;
- белые/молочные карточки;
- тёмно-синий основной текст;
- коралловый как один акцент;
- тёмные карточки только для важных фокусных блоков;
- UI должен выглядеть как премиальный рабочий кабинет мастера, а не игрушечная CRM.
