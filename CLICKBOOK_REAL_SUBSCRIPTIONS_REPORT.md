# ClickBook — профиль, каналы, переносы, отзывы

## Что исправлено

1. В профиле мастера активный переключатель `Онлайн / По адресу` теперь использует общий стиль основных кнопок ClickBook (`cb-neutral-primary`), а не отдельный чёрно-белый стиль.
2. После сохранения профиля появляется зелёное подтверждение «Изменения сохранены».
3. В сводке профиля добавлены две отдельные строки:
   - «Персональная страница»;
   - «Для отзывов».
4. Кнопка «Получить в Telegram» на экране успешной заявки теперь визуально такая же спокойная, как «Получить в VK».
5. При создании записи клиент больше не получает кнопки «Подтвердить / Перенос». Эти кнопки отправляются только в напоминании за 2 часа.
6. Напоминание за 24 часа стало информационным: без кнопок подтверждения/переноса.
7. Добавлена связанная логика переноса:
   - клиент за 2 часа нажимает «Перенос»;
   - запись отменяется, слот освобождается;
   - в чатах появляется жёлтая плашка предупреждения;
   - мастер предлагает новый слот из чата;
   - клиент получает Telegram/VK-кнопки «Подтвердить перенос» / «Не подходит»;
   - при подтверждении перенос записывается в Supabase и запись снова становится `confirmed` с новым временем;
   - если не подходит — предупреждение остаётся открытым, мастер подбирает другой слот.
8. В CRM-карточке клиента в истории визитов добавляется предупреждение «Был перенос».

## Новая миграция Supabase


```sql

```
