# Demo calendar/time range pass

- Calendar timeline is fixed to 08:00–00:00 and no longer expands to 01:00 when the browser time is outside work hours.
- 24:00 is displayed as 00:00 at the end of the day.
- Demo mode now has richer bookings, clients, services, availability, templates, notifications, favorites and chat threads.
- Stats/dashboard/clients/chats/availability use the same demo dataset through `useOwnedWorkspaceData`, so charts and lists are populated in demo mode.
