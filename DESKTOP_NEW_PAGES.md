# КликБук Desktop — новые внутренние страницы

В этой сборке основной сайт не трогается: `/`, `/login`, `/dashboard` и публичные страницы остаются как были.

Для Electron добавлена отдельная ветка внутреннего продукта:

- `/desktop/dashboard` — полноценный рабочий дашборд мастера;
- `/desktop/schedule` — расписание с недельным таймлайном, статусами, свободными слотами и загрузкой;
- `/desktop/clients` — CRM-клиенты, сегменты, таблица, профиль клиента;
- `/desktop/chats` — полноценная зона чатов: список диалогов, активный чат, карточка клиента и быстрые действия.

Electron теперь по умолчанию открывает:

```bash
http://localhost:3000/desktop/dashboard
```

Запуск:

```bash
npm install
npm run desktop:dev
```

Сборка Windows-приложения:

```bash
npm run desktop:dist
```

Если Windows ругается на `Cannot create symbolic link`, включите Windows Developer Mode или запустите терминал от администратора.

## Важно

Сейчас desktop-страницы работают на качественных demo-данных, без авторизации. Это сделано специально, чтобы можно было спокойно тестировать интерфейс приложения. Боевой сайт и старая логика кабинета не удалены.
