# Vexa v8 UI fixes

Сделано в этой сборке:

- Убран верхний заголовок Vexa-страниц над рабочим контентом.
- Скрыт переключатель `Демо / Рабочий` внутри Vexa-кабинета.
- Предупреждение про старую историю Telegram выделено отдельным аккуратным блоком для светлой и темной темы.
- Кнопки переведены на пурпурный брендовый цвет логотипа; hover-эффект сделан без обрезания на длинных кнопках.
- Добавлены мягкие hover-состояния на кликабельные элементы Vexa в светлой и темной теме.
- Список поисков получил пагинацию и ограничение по высоте, чтобы не уходить ниже конструктора поиска.
- Конструктор поиска получил token/chip-представление ключевых фраз и минус-слов.
- Блок качества фильтра адаптирован под темную тему.
- Источники теперь открываются кликом по всей строке.
- Добавлены темы источников: можно создавать темы и фильтровать источники по теме.
- Тексты источников приведены к правильной логике: пользователь добавляет Telegram-источник, а наш мониторинг/Vexa-бот читает оттуда новые сообщения и ищет совпадения.
- Аналитика усилена: insight-карточки, графики, таблица рабочих сценариев, рекомендации по источникам/фильтрам/доставке.
